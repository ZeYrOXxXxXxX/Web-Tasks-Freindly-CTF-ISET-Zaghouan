The Race Against Time

A High-Security Digital Vault

Mission Briefing

The vault is protected by a key that is constantly changing. The key is a number between 1 and 1000. If you guess the correct key, the vault unlocks, but only for a moment.

How to Proceed:

  1. Submit your guess to the endpoint: /guess?key=<your_guess>.

Warning: Any incorrect guess will cause the system to generate a new key, forcing you to start over.
